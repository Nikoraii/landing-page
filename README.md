## Getting Started

NPM example:
- Clone this project: `git clone {project-name}`
- Move to the directory: `cd {project-name}`
- Instal dependencies: `git i`
- Run: `npm run dev`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
